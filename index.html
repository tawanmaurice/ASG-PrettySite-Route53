What is this project?

This project shows that I can design and deploy a real, production-style environment on AWS — not just a single EC2 instance. The app runs behind a load balancer, scales with demand, and uses Route 53 for friendly DNS.

Key Features

Auto Scaling Group

EC2 instances launch and scale automatically based on demand.

Application Load Balancer

Distributes traffic across healthy servers for availability.

Route 53 DNS

Provides a clean domain name that maps to the ALB endpoint.

Terraform

Infrastructure as code — easy to rebuild, update, and destroy.

High-Level Architecture

  • VPC with public/private subnets
  • Internet Gateway + NAT Gateway
  • Application Load Balancer
  • Auto Scaling Group with Launch Template
  • Route 53 A record alias → ALB
